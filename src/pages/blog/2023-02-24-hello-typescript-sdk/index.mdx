## Scratch:

```bash
npx create-react-app orbital-hello-world --template typescript
cd oribtal-hello-world
npm start
```

Install the Orbital ts dependency:

```bash
npm install @orbitalhq/orbital-client@0.1.0-alpha.1
```

TODO: Have the films demo stack ready to deploy in a docker-compose.

Create the orbital config file: `orbital.config.json`, in the same folder as our `package.json`

```json
{
  "schemaServerUrl": "http://localhost:9022"
}
```

Then run:

```
npm run orbital:taxonomy:generate
```
