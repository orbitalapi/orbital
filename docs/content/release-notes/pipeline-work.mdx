
## Breaking changes

### Changes to customized pipeline runner apps
If you're running a customized pipeline runner, you'll need to make the following code changes:

#### Add new import:

```kotlin
@Import(HttpAuthConfig::class, PipelineStateConfig::class)
class PipelineRunnerApp
```

#### Breaking signature change for PipelineTransportBuilder interface

`PipelineTransportBuilder` (and the two subtypes: `PipelineInputTransportBuilder` and `PipelineOutputTransportBuilder`)
now take an additional parameter: `Pipeline`.

The updated signature is:

```
fun build(spec: S, logger: PipelineLogger, transportFactory: PipelineTransportFactory, pipeline: Pipeline): T
```
